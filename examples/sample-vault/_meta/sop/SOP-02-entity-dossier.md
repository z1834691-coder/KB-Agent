# SOP-02 · 实体档案八板块规范（示例）

> topic 无关的可复用 SOP 占位。真实版本定义实体档案的八大板块与"每条事实带信源"红线。

板块：速览 / 技术路线 / 产品矩阵 / 商业化 / 组织与人 / 战略叙事 / 风险争议 / 面试角度。

红线：禁止为凑行数注水；宁可少写，不可假写；每条关键事实带 tier + accessed_at。
