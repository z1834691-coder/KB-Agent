# SOP-01 · Frontmatter 与双链规范（示例）

> topic 无关的可复用 SOP 占位。真实版本定义：必填 frontmatter 字段、双链命名、时效策略字段。

- 必填字段：title / type / tags / status / last_reviewed / maintainer / related。
- 快变类型（如 `product_version_data`）必须带 `stale_after`。
- 双链目标必须存在，禁止悬空 `[[link]]`。
