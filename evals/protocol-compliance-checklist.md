# Protocol Compliance Checklist v0.1

每轮 run report 末尾必须勾选。

## Context

- [ ] 已读 README / DECISIONS / rubric v2 / failure patterns。
- [ ] 已读知识库总览或目标文件索引。
- [ ] 未一次性读取完整知识库。
- [ ] 动态上下文已记录。
- [ ] 长文已压缩为“事实摘要 / 判断依据 / 待确认点 / 最大泛化的最短程序”。
- [ ] 若上下文过载，已 compact 或开新 run。

## Tools

- [ ] 修改前已读目标文件相关片段。
- [ ] 工具调用风险等级已判断。
- [ ] 外部来源记录 URL、source_type、accessed_at。
- [ ] 没有把 web 搜索结果直接当事实入库。
- [ ] 写入后运行必要验证。
- [ ] git diff / status 已检查。

## Human Review

- [ ] 已检查是否触发 H2/H3/H4。
- [ ] 删除、批量移动、公开发布、版权敏感、目标公司风险均未绕过用户。
- [ ] 用户覆写已记录为 flywheel 数据。

## Eval

- [ ] actor 只看 actor-pack。
- [ ] gold 只给 Judge。
- [ ] 若校准集已确认，Judge 已先跑校准。
- [ ] 高分/低分 case 已抽查。
- [ ] 新 failure pattern 已追加或确认无需追加。
